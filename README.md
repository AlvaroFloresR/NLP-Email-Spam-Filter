# NLP Email Spam Filter (Classification)
Email spam filter using Naive-Bayes for Natural Language Processing (NLP)

## Highlights
- Naive-Bayes for Classification
- Count Vectorization
- Very high Accuracy (relying on correct grammar)

## How to Run
- Clone Repo
- Run `main.ipynb`
